# Data Science Projects Collection

## Overview

This repository contains a collection of end-to-end Data Science and Machine Learning projects that demonstrate practical applications of data analysis, predictive modeling, natural language processing, recommendation systems, and fraud detection. Each project follows a complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model development, evaluation, and visualization.

These projects are designed to strengthen real-world problem-solving skills and showcase proficiency in Python, SQL, machine learning, and data visualization tools.

---

## Projects Included

### 1. Customer Churn Prediction

Developed a classification model to predict customer churn using telecom customer data. The project includes data cleaning, feature engineering, exploratory data analysis, model training with Logistic Regression and Random Forest, and performance evaluation using ROC-AUC, F1-Score, and feature importance.

**Skills:** Classification, Feature Engineering, Logistic Regression, Random Forest, ROC-AUC, EDA

---

### 2. House Price Prediction

Built regression models to estimate house prices using housing datasets. The workflow includes handling missing values, outlier detection, feature scaling, categorical encoding, and comparing Linear Regression, Random Forest, and XGBoost using RMSE and R² metrics.

**Skills:** Regression, Feature Scaling, Data Preprocessing, Linear Regression, Random Forest, XGBoost

---

### 3. Sentiment Analysis on Social Media Tweets

Implemented an NLP-based sentiment classification system that categorizes tweets into Positive, Negative, and Neutral sentiments. The project includes text preprocessing, TF-IDF vectorization, machine learning classification, and visualization through word clouds and sentiment distributions.

**Skills:** Natural Language Processing, Text Preprocessing, TF-IDF, Logistic Regression, Naive Bayes, Data Visualization

---

### 4. Credit Card Fraud Detection

Developed a fraud detection system capable of identifying fraudulent credit card transactions. The project explores class imbalance, applies resampling techniques when appropriate, trains multiple classification models, and evaluates performance using Precision, Recall, F1-Score, and Confusion Matrix.

**Skills:** Fraud Detection, Classification, Imbalanced Learning, SMOTE, Random Forest, XGBoost

---

### 5. Movie Recommendation System

Built a recommendation engine using the MovieLens dataset by implementing Content-Based Filtering, Collaborative Filtering, and Matrix Factorization (SVD). The system generates personalized movie recommendations based on user preferences and movie similarity.

**Skills:** Recommendation Systems, Collaborative Filtering, Content-Based Filtering, Cosine Similarity, Matrix Factorization, SVD
